Этот сайт создан на Tilda
Существует множество различных инструментов разработчика. Например WordPress, который часто сравнивают с Tilda.
Для меня самое важное - это возможность достаточно быстро создать скелет сайта, без украшений, на который потом нужно будет добавить элементы: всякие кнопки, эффекты и изображения. Все эти украшения приходится писать самостоятельно с помощью кода HTML, свойств CSS, применяя JS ну и иногда, подглядывая в шаблоны.

Tilda это инструмент разработчика, который позволяет
оптимизировать затраты времени для разработки WEB продукта.
Возможны два различных варианта.
-- вы самостоятельно готовы администрировать свой интернет ресурс. Скорее всего, вы сами захотите разместить сайт на хостинге Tilda или WordPress.
-- сайт не нуждается в дальнейшем администрировании. Любой хостинг на выбор.
В результате вы получите готовый к использованию продукт..Его уникальность - это главная задача. А в какой "коробке" будете переносить готовое изделие не так и важно. По вашему желанию: в виде блоков или в виде кода.
